Manga Search App

Find something new to read.

Use the search bar below to search for your favorite Manga! Click the button under each cover to visit the MyAnimeList page for that manga to learn more about it. Please note that due to some API limitations certain English searches will yield unexpected results.
Whenever you search for something new the existing list will reset. You can also reset the list manually by using the "Clear the List" button.
